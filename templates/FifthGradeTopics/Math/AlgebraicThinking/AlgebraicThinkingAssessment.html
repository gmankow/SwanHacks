Algebraic Thinking — Exam

Part A: Writing Expressions

Write an algebraic expression for each situation.

  1. "A number increased by 11."
  2. "Six less than a number."
  3. "Triple a number, then add 4."
  4. "Half of a number."
  5. "A number multiplied by 9."

Part B: Evaluating Expressions

Substitute the value given for the variable.

  1. If x = 8, evaluate:
    • x + 12 = ____
    • 5x = ____
  2. If n = 15, evaluate:
    • n − 9 = ____
    • n ÷ 3 = ____
  3. If a = 6, evaluate:
    • 3a + 2 = ____
    • 4(a − 1) = ____

Part D: Solving One-Step Equations

Solve each equation. Show your work.

  1. x + 9 = 25
  2. m − 6 = 18
  3. 7k = 63
  4. p ÷ 4 = 5

Part E: Properties of Operations

Identify the property shown.

  1. 9 + 2 = 2 + 9
  2. (6 + 4) + 3 = 6 + (4 + 3)
  3. 5(n + 2) = 5n + 10

Part F: Simplifying Expressions

Use properties and combine like terms.

  1. 3x + 5 + x = ____
  2. 4(2 + n) = ____
  3. 2a + 7 − a = ____

Part G: Word Problems

Write an equation and solve.

  1. A number plus 13 equals 41.
    — Write the equation.
    — Solve.
  2. You earn $5 for each chore. After doing c chores, you earn $40.
    — Write the equation.
    — Solve for c.
  3. A number multiplied by 6 is 84.
    — Write the equation.
    — Solve.
  4. Zoe is 4 years older than Ben. Together their ages total 30.
    Let b = Ben's age.
    — Write an equation.
    — Solve for both ages.

Part H: Challenge Problems

Explain your reasoning.

  1. A student says that in the expression 4x + 7, “x must be multiplied first because variables always go first.”
    Explain whether this is correct.
  2. Two expressions are equivalent: 3(n + 5) and 3n + 15.
    Explain why they represent the same quantity.
