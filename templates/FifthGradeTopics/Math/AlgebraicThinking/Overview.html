Objectives

Students will understand and apply:

  • Writing, interpreting, and evaluating numerical and algebraic expressions
  • Using variables to represent unknowns in equations and real-world situations
  • Recognizing and generating numerical patterns and rules
  • Solving one-step equations using all four operations
  • Understanding the relationship between input–output tables, patterns, and expressions
  • Using properties of operations to simplify expressions (commutative, associative, distributive)

Key Content

1. Understanding Variables & Expressions

  • Introduce the idea that a variable is a symbol (like x or n) that represents an unknown or changeable value.
  • Teach how to:
    • Read and write algebraic expressions
    • Use operations with variables (x + 4, 6 – n, 3m, b ÷ 2)
    • Distinguish between expressions (no equals sign) and equations (includes =)
    • Use real-world examples:
      • "n + 5" could be 5 more than the number of stickers
      • "2x" could be twice as many hours, pages, apples, etc.

2. Evaluating Expressions

  • Teach students how to substitute values into expressions.
  • Examples:
    • If x = 3, evaluate 4x → 12
    • If n = 7, evaluate n – 2 → 5
  • Model step-by-step substitution and emphasize order of operations.
  • Encourage explaining what the expression represents in words.

3. Patterns & Rules

  • Explore numerical patterns using:
    • Input/output tables
    • Sequences
    • "Rule machines"
  • Teach students to:
    • Identify the pattern or rule
    • Complete the table
    • Represent the pattern using expressions (e.g., "multiply by 4 then add 2")
  • Show how patterns relate to graphs on coordinate planes as a preview to later algebra.

4. Solving One-Step Equations

  • Teach that an equation shows equality and can be solved.
  • Students learn inverse operations:
    • Addition ↔ Subtraction
    • Multiplication ↔ Division
  • Examples:
    • x + 5 = 12
    • m – 3 = 10
    • 4k = 20
    • p ÷ 2 = 6
  • Teach isolating the variable by "undoing" the operation.
  • Emphasize checking the solution by substituting back into the equation.

5. Properties of Operations

  • Teach the three key properties used to simplify expressions:
    • Commutative Property (order does not change the sum/product)
      a + b = b + a
      ab = ba
    • Associative Property (grouping does not affect sum/product)
      (a + b) + c = a + (b + c)
    • Distributive Property
      a(b + c) = ab + ac
  • Use visual models to show how these properties work.
  • Practice applying them to simplify or rewrite expressions.

6. Writing and Interpreting Real-World Equations

  • Teach students to turn real situations into equations:
    • "A number increased by 6 is 14" → x + 6 = 14
    • "Three groups of a number equals 21" → 3n = 21
  • Students solve the equation and interpret the meaning of the variable.
  • Provide word problems where students must:
    • Define variables
    • Write equations
    • Solve
    • Explain what their answer represents
